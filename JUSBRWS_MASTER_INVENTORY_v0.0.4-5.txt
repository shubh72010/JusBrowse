- data/repository/
  - BookmarkRepository.kt: Abstracts bookmark data logic.
  - HistoryRepository.kt: Abstracts history data logic.
  - DownloadRepository.kt: Abstracts download data logic.
  - PreferencesRepository.kt: Manages all user settings via DataStore (DNS, Engines, UI Prefs).
  - SiteSettingsRepository.kt: Abstracts per-site rule logic.

IV. SECURITY LAYER (The 12-Layer Defense)
------------------------------------------
- security/
  - ContentBlocker.kt: Multi-stage ad/tracker filter with CNAME uncloaking.
  - DnsResolver.kt: Secure DoH (DNS-over-HTTPS) resolution.
  - ApiScanner.kt: VirusTotal & Koodous integration for remote file scanning.
  - SecurityScanner.kt: Local threat detection for URLs.
  - ContainerManager.kt: Context-based WebView profile isolation.
  - DownloadValidator.kt: Validates file integrity and origin.
  - DownloadReceiver.kt: Intercepts system download broadcasts for safety checks.
  - CoherenceValidator.kt: Fingerprint consistency checker (Pillar 6 of Boring Engine).
  - FailsafeManager.kt: Emergency UI restoration if security hooks fail.
  - SuspicionScorer.kt: Aggregates detection events to score site "hostility".

V. PRIVACY LAYER (Anti-Fingerprinting)
---------------------------------------
- security (Privacy Engines)/
  - BoringEngine.kt: "Head Tracking" engine. Stable, session-locked drift.
  - Priv8Engine.kt: High-entropy API masking (The original privacy core).
  - RLEngine.kt: Recursive Logic engine for deep proxy spoofing.
  - FingerprintingProtection.kt: Base JS injection scripts for standard protection.
  - FakeModeManager.kt: Global persona controller (Switching hardware IDs).
  - FakePersona.kt: Metadata for specific device personas (iPhone, Pixel, etc.).

  - PrivacyBus.kt: Coordinates data flow between privacy engines.
  - PrivacyPacket.kt: Data container for sanitized hardware telemetry.
  - PrivateBrowsingManager.kt: State management for incognito-style sessions.
  - FollianBlocker.kt: Specialized blocker for modern "soft-tracking" scripts.
