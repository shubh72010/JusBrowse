# Security Policy

## Overview
JusBrowse is a privacy-focused browser developed under the JusDots project.  
Security and user privacy are core principles of the project.

If you discover a security vulnerability, please report it responsibly so it can be fixed before public disclosure.

---

## Supported Versions

Security updates are only guaranteed for the latest version of JusBrowse.

| Version | Supported |
|--------|-----------|
| Latest | ✅ Yes |
| Older versions | ❌ No |

Users are strongly encouraged to update to the latest release.

---

## Reporting a Vulnerability

If you discover a security vulnerability in JusBrowse, please report it **privately**.

Use GitHub’s **Private Vulnerability Reporting** feature to submit a report directly to the maintainers.

Please include the following information:

- Description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Screenshots, logs, or proof-of-concept code (if available)
- Suggested mitigation (optional)

Providing clear reproduction steps helps us resolve issues faster.

## Responsible Disclosure

Please follow responsible disclosure guidelines:

- **Do not publicly disclose the vulnerability** before it has been fixed.
- Give maintainers reasonable time to investigate and release a patch.
- Avoid exploiting the vulnerability outside of controlled testing.

We aim to acknowledge reports quickly and work toward a fix as soon as possible.

---

## Security Scope

Security issues may include (but are not limited to):

- Privacy leaks
- Fingerprinting bypasses
- WebView or rendering engine vulnerabilities
- Sandbox or permission bypass
- Network interception vulnerabilities
- Data storage leaks

Non-security bugs should be reported through the normal **GitHub Issues** page.

---

## Acknowledgements

We appreciate researchers and developers who responsibly report vulnerabilities and help improve JusBrowse security.

Contributors who report valid security issues may be acknowledged in release notes unless they prefer to remain anonymous.
